1.키워드란 무엇인가?
: 자바의 키워드는 예약어로서 프로그램의 제어 흐름, 선언, 클래스 식별자
식을 쓸 때 사용한다.

2.2의 보수법이란?
: 주어진 이진수의 모든 자리의 숫자를 반전(0을 1로, 1을 0으로)시킨 뒤 
여기에 1을 더하면 2의 보수를 얻을 수 있다.

3.컴퓨터에서 음의 정수 표현방법에 대하여 설명하시오.
: 양의 정수를 2진법으로 나타낸 뒤 2의 보수값을 더해 8bit보다 큰 
자릿수의 값은 버리고 음수를 표현한다.

4.-5가 컴퓨터에서의 2진수 값을 나타내시오.
: *정수 5의 2진법은 00000101, 여기에 1을 더해 2의 보수법 11111011로 변환
→ 11111011

5.실수에서 오차가 생기는 원인은 무엇인가?
:
두 실수 사이에 존재하는 숫자는 무한대인데, 메모리는 상대적으로 유한하기 때문에
가장 근사치의 값을 표현하도록 되어있다.

6.실수의 표현법(고정소수법, 부동소수법)을 설명하시오.
:
(고정소수법) - 정수를 표현하는 비트 수와 소수를 표현하는 비트 수를 미리 정해 놓고,
해당 비트 만큼만 사용해서 숫자를 표현하는 방식
(부동소수법) - 소수점의 위치를 고정하지 않고 그 위치를 나타내는 수를 따로 적는 
것으로, 유효숫자를 나타내는 가수와 소수점의 위치를 풀이하는 지수로 나누어 표현한다.
IEEE 754는 전기 전자 기술자 협회(IEEE)에서 개발한 컴퓨터에서 부동소수점을
표현하는 가장 널리 쓰이는 표준이다.

7.정밀도란 무엇인가요?
:
정밀도(precision)는 변수가 얼마나 많은 유효 숫자를 가질 수 있느냐를 말하는 것이다.

8.상수란 무엇이며 변수와의 차이는?
: 변수와 반대되는 개념으로 final로 지정한 초기값에서 변하지 않는 
값을 가리킨다.

9.초기화란 무엇인가?
: 객체를 선언하고 값을 최초로 할당하는 것이다

10.리터럴이란 무엇인가요?
: 자료형을 기반으로 표현이 되는 상수를 의미한다. 
(정수형 리터럴과 실수형 리터럴로 구분한다.)

11.아래의 내용이 에러가 나는 이유와 수정방법은?
System.out.println(3147483647 + 3147483648);

: 연산값이 int로 표현할 수 있는 범위보다 크기 때문에 l이나 L을
뒤에 붙여서 long형으로 변환해준다. 

12.자동 형변환과 명시적 형변환에 대하여 설명하시오.
:
서로 다른 형태의 변수값이 있을 때, 자동으로 정수는 더 큰쪽으로 실수는 소수를 보전
하는 쪽으로 형태를 동일하게 맞추게 된다. 이를 자동 형변환이라 한다. 
이 때, 개발자가 원하는대로 변수의 형태를 지정할 수 있는데 이를 명시적 형변환이라 한다. 

13.아래의 에러가 나는 이유를 설명하고 수정하시오.
    	int a = 3;
    	int b = 4;
    	
    	double result = a / b;
    	
    	System.out.println(result);

:
int는 정수를 나타내는 형인데 실수의 형태인 double로 result값을 요청
했으므로 소수 부분을 생략하고 출력한다.
그러므로 명시적 형변환을 통해 a 혹은 b에 double을 명령하거나 int값 a 
초기화를 double로 수정한다면 에러가 해결된다.  

14.이스퀘이프 시퀀스의 종료를 나열하고 기능을 설명하시오.
:
화면상의 어떤 상황 또는 상태를 표현하기 위해 약속된 문자
'|b' → 백스페이스 문자
'|t' → 탭 문자
'|n' → 개행 문자
'|r' → 캐리지 리턴 문자
'||' → 백슬래시 문자
'|'' → 작은따옴표 문자
'|"' → 큰따옴표 문자
